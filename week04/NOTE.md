## 有限状态机

+ 每个状态都是一个机器
  + 每个机器都可以进行计算、存储、输出...
  + 所有机器接受的输入必须一致
  + 每个机器本身没有状态，如果用函数来表示，必须没有副作用
+ 每个机器知道下一个状态是什么
  + 有确定的下一个状态（Moore）
  + 根据输入决定下一个状态（Mealy）



```js
// JS中的有限状态机（Mealy）

// 每个函数是一个状态
function state(input)
{
  // ...处理逻辑
  return next; // 返回下一个状态
}

// 调用
while(input) {
  // 获取输入
  state = state(input) // 把状态机的返回值作为下一个状态
}
```
